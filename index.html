Visual Studio Code is a freeware source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git. Users can change the theme, keyboard shortcuts, preferences, and install extensions that add additional functionality. If you want to download VSCode the link is in the console screen.

Just right click anywhere a menu will pop up in the bottom click on inspect. When you click on the inspect on the right side a source page will come in the top of the source. On the top of the source page You will see 5 texts Element,Console,Sources,Network and Performance. Click on console to view the colsole page.
